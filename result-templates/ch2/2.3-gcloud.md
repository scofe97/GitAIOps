# 2.3 gcloud CLI 설치 — 예상 결과

## 상태 확인

```
$ gcloud config list
[core]
account = <USER>@gmail.com
project = <PROJECT_ID>

[compute]
region = asia-northeast3
zone = asia-northeast3-a
```

## 체크리스트

- [ ] gcloud CLI 설치됨
- [ ] GCP 인증 완료 (gcloud auth login)
- [ ] 프로젝트 ID 설정됨
- [ ] 기본 리전/존 설정됨

## 이 단계 이후 아키텍처

```
독자 PC → gcloud CLI → GCP 프로젝트 연결
```
