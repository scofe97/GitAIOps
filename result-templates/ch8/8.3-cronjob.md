# 8.3 배치 자동화: CronJob — 예상 결과

## 상태 확인

```
$ kubectl get cronjob -n notiflex
NAME                    SCHEDULE      SUSPEND   ACTIVE   LAST SCHEDULE
notiflex-healthcheck    */5 * * * *   False     0        1m

$ kubectl get jobs -n notiflex
NAME                              COMPLETIONS   DURATION   AGE
notiflex-healthcheck-xxxxxxxxxx   1/1           5s         1m

$ kubectl logs job/notiflex-healthcheck-xxxxxxxxxx -n notiflex
Health check passed (HTTP 200)
Response: {"status":"ok"}
```
