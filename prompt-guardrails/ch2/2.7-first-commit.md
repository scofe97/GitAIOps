### 단계 1: JOURNEY.md 생성

`prompt-guardrails/shared/journey-template.md`를 복사하여 `notiflex-platform/JOURNEY.md`를 생성한다.

지금까지 만든 모든 파일을 GitHub에 커밋하고 푸시한다.

```bash
cd notiflex-platform
git add .
git commit -m "Initial commit: Notiflex app, K8s manifests, CLAUDE.md, JOURNEY.md"
git push origin main
```
